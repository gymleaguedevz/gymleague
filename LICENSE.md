### ❌ Inf Stamina stops working mid-session

**Cause:** The stamina loop thread was killed by a server-side timeout.
**Fix:**
- Toggle the Inf Stamina feature OFF, wait 3 seconds, then toggle it back ON
- If it still fails, reduce `GL_FARM_DELAY` slightly to re-sync the loop timing

---

### ❌ Script crashes on Android

**Cause:** Insufficient RAM or outdated executor APK.
**Fix:**
1. Close all background apps before running
2. Update your mobile executor to the latest APK
3. Reduce active feature count — run only Auto Farm + Anti-AFK if performance is poor

---

### ❌ iOS script not injecting

**Cause:** Developer profile trust expired or Roblox was updated.
**Fix:**
1. Go to **Settings → General → VPN & Device Management**
2. Re-trust the developer profile
3. Delete and reinstall the executor if trust fails
4. Ensure Roblox is fully up to date via the App Store

### ❌ Money farm not generating income
